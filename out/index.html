index.js

This file sets up express application, the server and implements the various calls to the API endpoints. The Data use mongoose models to structure the data, that can be found in the models file. Access is established through authentication which is implemented using Passport and can be located in the passport file. Once connected the data is accessed via the MongoDB Atlas. The application is hosted on Heroku.
