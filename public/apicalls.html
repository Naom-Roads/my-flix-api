Business Logic URL HTTP Method Request Format Response Format
Get List of Movies for User /movies GET None A JSON object holding data about all of the movies
Get data about a single movie /movies/:title GET None A JSON object holding data about a single movie, containing an ObjectID, name, image, description, genres, director, image and whether the movie is featured 
                
                {
                 ObjectID: 
                 title: "Embrace of the Serpent" 
                 description: 
                 director: 
                 genre: 
                 featured:
                 image: 
               }
Get data about director from a single movie using movie id "/movies/:id/director" GET None A JSON object holding data about a single director(containing an name, bio, birth year, death year) 
                
                {
                 name: "Ciro Guerra"; 
                 bio: "Ciro Guerra was born on February 6th 1981 in Rio de Oro, Cesar, Colombia"
                 birthyear: 1981 - Present; 
                 deathyear: (If applicable)
               }
Get data about a genres from single movie id /movies/:id/genres GET None A JSON object with a list of a movie's genres including name and description of each genre 
                
                {
                 genre: "Thriller"
                 description: "Action Packed"
               }
Add a Movie /movies POST A JSON Object holding data about the movie to add, in this structure: 
                
                {   title: req.body.title,
                    description: req.body.description,
                    director: req.body.director,
                    genres: req.body.genres,
                    imageurl: req.body.imageurl,
                    featured: req.body.featured,
                }
A JSON object holding data about the movie that was added, including ObjectID: 
                    
                    {
                        _id: 
                        title: req.body.title,
                        description: req.body.description,
                        director: req.body.director,
                        genres: req.body.genres,
                        imageurl: req.body.imageurl,
                        featured: req.body.featured,
                   }
Allow new user to register /users POST A JSON Object holding data about the user to add structured like: 
                    
                        {
                            Username: req.body.username,
                            Password: req.body.password,
                            Email: req.body.password,
                            Email: req.body.email,
                            Birthday: req.body.birthday,
                        }
A JSON object holding data for new user with ObjectID included 
                
                    {
                        _id: 
                        Username: req.body.username,
                    Password: req.body.password,
                    Email: req.body.password,
                    Email: req.body.email,
                    Birthday: req.body.birthday,
                    }
Allows users to update their profile username users/:username PATCH None A text message indicating that their movie list has been updated
Allows users to add a movie to their list of favorites /users/:username/movies/:movieID PUT Inserts movie into json document containing favoritemovies Response Format
Remove a Movie by ID /movies/[id] DELETE None A text message indicating whether the student was succussfully removed
Allows users to remove a movie from their list(showing that a movie has been removed) /users/[id]/movies/[title] DELETE Request Format A text message indicating the movie was removed from the list
Allows users to deregister(showing a text that the user email has been removed) URL DELETE Request Format A text message indicating user email has been removed
