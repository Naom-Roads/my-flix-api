Overview

Api allows users to pull information on various movies and add them to their profile MongoDB is used for the database and Mongoose is used for business logic. The API can be tested using postman. It adheres to data security regulations.

You will be able to pull the following:
  • Film Name
  • Actor Names
  • Release Dates
  • Movie Ratings
  • Director Name
  • Country of Origin
